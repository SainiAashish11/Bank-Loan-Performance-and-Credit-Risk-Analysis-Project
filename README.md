# Bank Loan Performance and Credit Risk Analysis

## 📌 Project Overview

This project analyzes bank loan data to evaluate lending performance, borrower risk, and overall portfolio health. The objective is to transform raw loan data into actionable insights that support data-driven decision-making in a banking and financial services context.

The solution follows an industry-aligned workflow using SQL for data validation and Power BI for interactive visualization.

## 🎯 Objectives

- Analyze loan applications, disbursements, and repayments
- Assess portfolio health using Good vs Bad loan classification
- Identify lending trends, borrower behavior, and risk patterns
- Enable drill-down analysis for loan-level investigation

## 🗂 Dataset Overview
  
- Each row in the dataset represents a unique loan account.

### Key attributes include:

- Loan details: loan amount, interest rate, term, installment
- Borrower profile: employment length, income, home ownership
- Risk indicators: grade, sub-grade, debt-to-income (DTI)
- Loan status: Fully Paid, Current, Charged Off
- Key dates: loan issue date and repayment dates

## 🔍 Key Insights
- Majority of loans are classified as Good Loans, indicating a healthy portfolio
- Charged-off loans show higher interest rates and DTI, highlighting higher credit risk
- Loan demand is driven primarily by debt consolidation and credit card purposes
- Consistent monthly trends indicate stable credit demand

## 🛠 Tools & Technologies
- Power BI – Data visualization and dashboard development
- SQL – Data validation and KPI verification
- Excel / CSV – Source data handling

## 📈 Outcome
The dashboard enhances visibility into loan performance, strengthens risk assessment, and supports strategic decision-making by transforming raw loan data into meaningful insights.
